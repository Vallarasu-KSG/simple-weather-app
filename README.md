# 🌤️ Weather App

This **Weather App** helps you quickly and easily get real-time weather information for any city worldwide using the [OpenWeatherMap API](https://openweathermap.org/api).

---

## 🚀 Features

- 🔍 **City Search**: Type the name of any city and click **Search** or press **Enter** to get live weather updates.
- 🌡️ **Current Temperature**: Displays temperature in Celsius.
- 💧 **Humidity**: Shows the current humidity percentage.
- 🌬️ **Wind Speed**: Displays wind speed in kilometers per hour (km/h).
- 🌈 **Weather Icons**: Visually shows weather conditions (Clear, Rainy, Cloudy, etc.).
- ⚠️ **Automatic Error Handling**: Shows a clear message if the city name is invalid or input is empty.
- 📱 **Mobile Responsive**: Works seamlessly on both desktop and mobile screens.
- 🎨 **Clean UI**: Simple, colorful, and user-friendly design.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript (Vanilla JS)
- **API**: [OpenWeatherMap API](https://openweathermap.org/api)

---

## 🔧 How It Works

1. Enter a **city name** in the input field.
2. Press **Enter** or click the **Search** button.
3. Instantly view:
   - Temperature (°C)
   - Humidity (%)
   - Wind speed (km/h)
   - Weather condition with an icon
4. If the input is empty or the city is not found, a helpful error message will appear.

---

## 📦 Features in Detail

- ✅ Uses **Fetch API** to retrieve data from OpenWeatherMap.
- ✅ Parses JSON response and dynamically updates the UI.
- ✅ Automatically updates weather icons based on the condition.
- ✅ Responsive design ensures it works across all screen sizes.
